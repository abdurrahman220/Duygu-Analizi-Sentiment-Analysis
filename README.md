# Duygu-Analizi-Sentiment-Analysis
Python ve Scikit-learn ile geliştirilmiş e-ticaret duygu analizi (Sentiment Analysis) projesi. Müşteri yorumlarını Olumlu, Olumsuz ve Nötr olarak sınıflandırır. Etkileşimli GUI üzerinden anlık tahmin yapar ve detaylı makine öğrenmesi metrikleri (ROC AUC, MCC, F1) sunar. Veri dengesizliği N-gram teknikleriyle optimize edilmiştir.
